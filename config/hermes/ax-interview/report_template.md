## 4. 환자 진료정보 수집 항목 상세

> 아래 내용은 AI 에이전트가 자동으로 해당 업무를 취합/요약/추천하고 대시보드를 생성하기 위해 EMR, OCS 등에서 실시간으로 수집하고 분석해야 하는 상세 항목과 규칙 리스트입니다.

| 구분 | 조사 항목 | 내용 및 예시 |

| **진단** | 부진단 | [예: COPD] |
| **검사** | 진단검사 | slip별 Trend 조회에서 이상치 확인, 최근값으로 비교<br>*(예)* `[5.18] Routine CBC : 12.30 x10³/㎕, CRP 9.33mg/dL` |
| **검사** | 진단검사_미생물 | 미생물검사 환자별 결과 조회<br>*(예)* `Fungus Culture : Candida albicans` |
